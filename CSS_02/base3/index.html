Блок навигации

  • Ширина 15еm
  • Должен пристутствовать цвет фона
  • Распологается слева
  • Блок навигации наполните контентом достаточным, чтоюы по высоте он начинал захватытвать вторую статью, как показано в данном примере

Статья 1

Размер блока:
Занимает всю доступную ширину
Должен пристутствовать цвет фона
Во всех "статьях" сдлеайте отступ снизу от соседей равных 2em, при помощи свойства margin-bottom

Статья 2

Размер блока:
занимает всю доступную ширину
Должен пристутствовать цвет фона
Во всех "статьях" сделайте отступ снизу от соседей равным .2em припомощии свойства margin-bottom.
Картинка вставлена при помощи тега, плавание организовано по правому краю внутри данной статьи для этой картинки
Нижний блок, подвал. Занимает всю достпупную ширину. Высота определяется помещенным внутри контентом.
